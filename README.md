Программа предоставляет пользователю возможность взаимодействия через меню, в котором можно выбрать одно из действий:

Прямоугольник:
  Вычислить периметр
  Вычислить площадь
  Вычислить длину диагонали

Треугольник:
  Вычислить периметр
  Вычислить площадь по формуле Герона
  Проверка на равнобедренность

Окружность:
  Вычислить длину окружности
  Вычислить площадь
  Вычислить площадь кругового сектора
